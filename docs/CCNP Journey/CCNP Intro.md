# CCNP Enterprise Certification Journey

For the last few months, I have been studying for Cisco's **CCNP ENCOR (Enterprise Core) 350-401 exam**. This exam is one of two that I will need to pass in order to achieve the **CCNP (Cisco Certified Network Professional)** certification.

The second exam I will need to pass is a **concentration exam**. Cisco provides a variety of concentrations to choose from. I chose **"Designing and Implementing Cloud Connectivity."** The associated exam for this concentration is the **ENCC 300-440**.

---

## Why the Cloud Concentration

I am choosing the **Cloud concentration** because I believe IT trends are pointing in that direction.

I have a little experience with cloud platforms, including:

- Hosting a website on **AWS S3**
    
- Managing **Microsoft Azure AD**
    
- Using **Google Cloud APIs**
    

However, I want to go deeper and better understand **how Cisco integrates with the various cloud platforms**.
