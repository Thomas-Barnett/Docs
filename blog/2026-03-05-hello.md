# Hello

This is my first blog post on this site.

The goal of this blog is to document things I learn while working with networking, infrastructure, and technology. Writing things down helps me better understand them, and hopefully it can help others as well.

## What you'll find here

Over time I plan to write about:

* Networking and Cybersecurity concepts
* Home lab experiments
* Project documentation
* Tools, workflows, scripts, and practices that I find useful
* Interesting problems I run into while working in IT
* Adjacent technical hobbies (electronics, automation, maker projects, etc.)
* Non-technical hobbies and interests outside of IT

## Why I write things down

One of the best ways to learn something is to explain it.
This blog serves as a place for me to organize my thoughts and record useful information for future reference.

More posts coming soon.
